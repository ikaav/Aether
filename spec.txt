# Aether - Complete Technical Specification

## 1. NAME & PURPOSE
**Aether** - Make habit initiation frictionless, automate gentle recovery, visually reinforce real consistency, and protect sleep, privacy, intrinsic motivation, and cognitive load.

## 2. PRIMARY GOALS (Ranked)
1. Maximize reliable starts and fast restarts, not brittle streaks
2. Prevent late-night rescue abuse and protect sleep
3. Keep privacy first, proofs optional and local
4. Default to low friction, passive inference, with opt-in escalation
5. Reward competence predictably, not gambling

## 3. DESIGN PRINCIPLES (Non-negotiable)
- Default to passive, local computation, minimal prompts
- Make automated changes reversible via weekly review
- Ceilings and floors change slowly; changes are predictable and explainable
- Proofs are opt-in, on-device by default, short retention
- Max active habits default = 3, unlock rules controlled
- Immediate, clear visual feedback for starts and completions
- No gambling mechanics, no paid advantages that affect score
- Bedtime and anti-shame are hard constraints

## 4. TARGET METRICS (Initial KPI Targets)
- Median open-to-start < 20 seconds
- Fraction of floors completed per user per week > 0.7 after 4 weeks
- Fraction of sessions continuing past floor > 0.4
- Median restart latency after a miss ≤ 1 day
- Nighttime salvage rate < 5% of completions

## 5. TECHNICAL ARCHITECTURE

### Tech Stack Recommendation
**Backend:**
- Language: Node.js with TypeScript or Go
- Database: PostgreSQL with Redis for caching
- API: REST with GraphQL consideration for complex queries
- Cloud: AWS/GCP with serverless functions for batch jobs
- Authentication: OAuth2 + JWT with refresh tokens
- Queue: Redis Bull or AWS SQS for background jobs

**Mobile:**
- iOS: SwiftUI with Core Data, CloudKit optional
- Android: Kotlin with Room database, WorkManager
- Shared: Protocol Buffers for API communication
- Analytics: Custom events with batch upload

**Desktop Helper (Optional):**
- Electron or native (Swift/C++) for window title monitoring
- Local-only processing with hashed outputs

### Database Architecture
**Primary Tables:**
- `users` - user profiles, preferences, encryption keys
- `habits` - habit definitions, scheduling, algorithm state
- `session_events` - all session activities, immutable log
- `token_buckets` - reward state per habit
- `floor_history` - audit trail of automatic adjustments
- `analytics_events` - telemetry with algorithm versioning

```typescript
class DeepLinkManager {
  async openResource(url: string): Promise<boolean> {
    try {
      if (await this.canOpenURL(url)) {
        await this.openURL(url);
        return true;
      } else {
```

### Multi-Device Timer Synchronization

**Conflict Resolution Rules:**
1. **Same habit, overlapping sessions:** Server accepts both, flags for user review
2. **Network partition recovery:** Merge by session_id deduplication
3. **Clock skew:** Use server timestamp as canonical, adjust local durations

```typescript
    pending_events: pending

    await this.resolveConflict(conflict);
```

```typescript
enum SessionState {
  IDLE = 'idle',
  STARTING = 'starting',
  FLOOR_RUNNING = 'floor_running',
  FLOOR_MET = 'floor_met',
  TARGET_RUNNING = 'target_running',
  COMPLETING = 'completing',
  COMPLETED = 'completed',
  SALVAGING = 'salvaging',
  REPAIRING = 'repairing',
  ERROR = 'error'
}

class SessionStateMachine {
  async transition(from: SessionState, to: SessionState, event: any): Promise<boolean> {
    // Validate transition
    if (!this.isValidTransition(from, to)) {
      throw new InvalidTransitionError(`Cannot transition from ${from} to ${to}`);
    }
    
    // Execute side effects
    await this.executeTransitionEffects(from, to, event);
    
    // Persist state change
    await this.persistStateChange(to, event);
    
    return true;
  }
  
  private async executeTransitionEffects(from: SessionState, to: SessionState, event: any) {
    switch (to) {
      case SessionState.FLOOR_RUNNING:
        await this.enableDND();
        await this.openDeepLink(event.resource_url);
        await this.startTimer(event.floor_seconds);
        break;
      case SessionState.FLOOR_MET:
        await this.awardToken();
        await this.triggerHaptic(HapticType.SUCCESS);
        await this.showContinuePrompt();
        break;
      // ... other cases
    }
  }
}
```

### Adaptive Floor Algorithm (Enhanced EMA)

```typescript
interface EMAConfig {
  alpha: number; // 0.2 default
  lowerThreshold: number; // 0.75
  raiseThreshold: number; // 0.9
  lowerDaysRequired: number; // 3
  raiseDaysRequired: number; // 7
  raiseFactor: number; // 1.05
  lowerFactor: number; // 0.9
  maxRaiseFrequency: number; // 7 days
  repairCooldown: number; // 72 hours
}

class FloorAdjustmentEngine {
  async processNightlyUpdates(userId: string): Promise<FloorAdjustment[]> {
    const habits = await this.getUserHabits(userId);
    const adjustments: FloorAdjustment[] = [];
    
    for (const habit of habits) {
      const todayCompletion = await this.getTodayCompletion(habit.id);
      const newEMA = this.updateEMA(habit.ema, todayCompletion);
      
      const adjustment = await this.evaluateFloorAdjustment(habit, newEMA);
      if (adjustment) {
        adjustments.push(adjustment);
        await this.logFloorHistory(habit.id, adjustment);
      }
      
      await this.updateHabitEMA(habit.id, newEMA);
    }
    
    return adjustments;
  }
  
  private async evaluateFloorAdjustment(habit: Habit, ema: number): Promise<FloorAdjustment | null> {
    const consecutiveLow = await this.getConsecutiveDaysBelow(habit.id, this.config.lowerThreshold);
    const consecutiveHigh = await this.getConsecutiveDaysAbove(habit.id, this.config.raiseThreshold);
    
    const now = new Date();
    const lastRepair = habit.last_repair_ritual_ts ? new Date(habit.last_repair_ritual_ts) : null;
    const lastRaise = habit.last_floor_raise_ts ? new Date(habit.last_floor_raise_ts) : null;
    
    // Lower floor if consistently below threshold
    if (consecutiveLow >= this.config.lowerDaysRequired) {
      const newFloor = Math.max(
        habit.min_floor_seconds,
        Math.round(habit.floor_seconds * this.config.lowerFactor)
      );
      
      if (newFloor < habit.floor_seconds) {
        return {
          habitId: habit.id,
          oldValue: habit.floor_seconds,
          newValue: newFloor,
          reason: 'ema_lower',
          emaSnapshot: ema
        };
      }
    }
    
    // Raise floor if consistently above threshold (with cooldowns)
    if (consecutiveHigh >= this.config.raiseDaysRequired) {
      const hoursSinceRepair = lastRepair ? (now.getTime() - lastRepair.getTime()) / (1000 * 60 * 60) : Infinity;
      const daysSinceRaise = lastRaise ? (now.getTime() - lastRaise.getTime()) / (1000 * 60 * 60 * 24) : Infinity;
      
      if (hoursSinceRepair > this.config.repairCooldown && daysSinceRaise > this.config.maxRaiseFrequency) {
        const targetCap = habit.target_seconds * 0.8; // Don't raise floor above 80% of target
        const newFloor = Math.min(
          targetCap,
          Math.round(habit.floor_seconds * this.config.raiseFactor)
        );
        
        if (newFloor > habit.floor_seconds) {
          return {
            habitId: habit.id,
            oldValue: habit.floor_seconds,
            newValue: newFloor,
            reason: 'ema_raise',
            emaSnapshot: ema
          };
        }
      }
    }
    
    return null;
  }
}
```

### Token System & Reward Engine

```typescript
interface RewardUnlock {
  type: 'cosmetic' | 'story' | 'soundtrack';
  assetId: string;
  rarity: 'common' | 'rare' | 'epic';
  unlockCost: number;
  metadata: Record<string, any>;
}

class RewardEngine {
  async processTokenAward(habitId: string, userId: string): Promise<TokenResult> {
    const bucket = await this.getTokenBucket(userId, habitId);
    
    if (bucket.current_tokens >= 4) {
      return { tokensAdded: 0, reason: 'at_capacity' };
    }
    
    // Check quiet period
    if (bucket.quiet_until && new Date() < bucket.quiet_until) {
      return { 
        tokensAdded: 1, 
        reason: 'quiet_period',
        tokensVisible: false 
      };
    }
    
    await this.incrementTokenBucket(bucket);
    const unlocks = await this.checkForUnlocks(bucket);
    
    return {
      tokensAdded: 1,
      newTotal: bucket.current_tokens + 1,
      unlocksTriggered: unlocks
    };
  }
  
  private async checkForUnlocks(bucket: TokenBucket): Promise<RewardUnlock[]> {
    const unlocks: RewardUnlock[] = [];
    
    if (bucket.current_tokens >= 2) {
      const cosmetic = await this.selectRandomCosmetic(bucket.habit_id);
      if (cosmetic) unlocks.push(cosmetic);
    }
    
    if (bucket.current_tokens >= 3) {
      const story = await this.selectStoryPostcard(bucket.habit_id);
      if (story) unlocks.push(story);
    }
    
    if (bucket.current_tokens >= 4) {
      const premium = await this.selectPremiumUnlock(bucket.habit_id);
      if (premium) unlocks.push(premium);
    }
    
    return unlocks;
  }
}
```

### Heat-Based Nudging System

```typescript
interface HeatSignals {
  recentStarts: number; // 0-1, recent session frequency
  calendarBusy: number; // 0-1, how busy today looks
  sleepDebt: number; // 0-1, estimated tiredness
  timeOfDay: number; // 0-1, based on personal patterns
  deviceUsage: number; // 0-1, recent app activity
}

class NudgeEngine {
  calculateHeatScore(signals: HeatSignals): number {
    const weights = {
      recentStarts: 0.25,
      calendarBusy: 0.20,
      sleepDebt: 0.25,
      timeOfDay: 0.15,
      deviceUsage: 0.15
    };
    
    return Math.min(1.0, 
      weights.recentStarts * signals.recentStarts +
      weights.calendarBusy * signals.calendarBusy +
      weights.sleepDebt * signals.sleepDebt +
      weights.timeOfDay * signals.timeOfDay +
      weights.deviceUsage * signals.deviceUsage
    );
  }
  
  async shouldShowNudge(userId: string, habitId: string): Promise<boolean> {
    const heat = await this.calculateCurrentHeat(userId);
    
    // Only nudge in the "sweet spot"
    if (heat < 0.2 || heat > 0.7) {
      return false;
    }
    
    // Respect user's DND and calendar
    if (await this.isUserBusy(userId)) {
      return false;
    }
    
    // Check nudge frequency limits
    if (await this.recentNudgesSent(userId, habitId) > 2) {
      return false;
    }
    
    return true;
  }
  
  generateNudgeMessage(habit: Habit, heat: number): string {
    const templates = [
      "Your {habit_name} lantern is ready to light 🏮",
      "Perfect time for {habit_name}. Start?",
      "Your {habit_name} space is prepared ✨"
    ];
    
    const template = templates[Math.floor(heat * templates.length)];
    return template.replace('{habit_name}', habit.name);
  }
}
```

## 10. OFFLINE SYNCHRONIZATION & CONFLICT RESOLUTION

### Offline Event Queue

```typescript
interface OfflineEvent {
  id: string;
  type: 'session_start' | 'session_complete' | 'token_claim' | 'setting_change';
  timestamp: Date;
  payload: any;
  syncStatus: 'pending' | 'synced' | 'failed' | 'conflict';
  retryCount: number;
  deviceId: string;
}

class OfflineSyncManager {
  private eventQueue: OfflineEvent[] = [];
  
  async queueEvent(type: string, payload: any): Promise<void> {
    const event: OfflineEvent = {
      id: this.generateUUID(),
      type: type as any,
      timestamp: new Date(),
      payload,
      syncStatus: 'pending',
      retryCount: 0,
      deviceId: await this.getDeviceId()
    };
    
    this.eventQueue.push(event);
    await this.persistQueue();
    
    // Try immediate sync if online
    if (await this.isOnline()) {
      this.syncPendingEvents();
    }
  }
  
  async syncPendingEvents(): Promise<SyncResult> {
    const pending = this.eventQueue.filter(e => e.syncStatus === 'pending');
    if (pending.length === 0) return { success: true, conflicts: [] };
    
    try {
      const response = await this.api.batchSync({
        device_id: await this.getDeviceId(),
        events: pending.map(e => ({
          client_id: e.id,
          type: e.type,
          timestamp: e.timestamp.toISOString(),
          payload: e.payload
        }))
      });
      
      // Process successful syncs
      for (const success of response.successful) {
        const event = pending.find(e => e.id === success.client_id);
        if (event) {
          event.syncStatus = 'synced';
        }
      }
      
      // Handle conflicts
      const conflicts: ConflictResolution[] = [];
      for (const conflict of response.conflicts) {
        const resolution = await this.resolveConflict(conflict);
        conflicts.push(resolution);
      }
      
      await this.persistQueue();
      return { success: true, conflicts };
      
    } catch (error) {
      // Mark events as failed for retry
      pending.forEach(event => {
        event.retryCount++;
        if (event.retryCount > 3) {
          event.syncStatus = 'failed';
        }
      });
      
      await this.persistQueue();
      throw error;
    }
  }
  
  private async resolveConflict(conflict: SyncConflict): Promise<ConflictResolution> {
    switch (conflict.type) {
      case 'session_overlap':
        return this.resolveSessionOverlap(conflict);
      case 'token_mismatch':
        return this.resolveTokenMismatch(conflict);
      case 'setting_conflict':
        return this.resolveSettingConflict(conflict);
      default:
        return { resolution: 'server_wins', reason: 'unknown_conflict_type' };
    }
  }
  
  private async resolveSessionOverlap(conflict: SyncConflict): Promise<ConflictResolution> {
    const localSession = conflict.local_data;
    const serverSession = conflict.server_data;
    
    // If sessions are within 5 minutes, likely same session from different devices
    const timeDiff = Math.abs(
      new Date(localSession.started_at).getTime() - 
      new Date(serverSession.started_at).getTime()
    );
    
    if (timeDiff < 5 * 60 * 1000) {
      // Merge sessions, prefer longer duration
      return {
        resolution: 'merge',
        merged_data: {
          ...serverSession,
          duration_seconds: Math.max(localSession.duration_seconds, serverSession.duration_seconds),
          floor_reached: localSession.floor_reached || serverSession.floor_reached,
          target_reached: localSession.target_reached || serverSession.target_reached
        }
      };
    }
    
    // Different sessions, keep both
    return { resolution: 'keep_both', reason: 'distinct_sessions' };
  }
}
```

```typescript
interface TimerState {
  sessionId: string;
  habitId: string;
  startedAt: Date;
  floorSeconds: number;
  currentPhase: 'floor' | 'target' | 'stretch';
  elapsedSeconds: number;
  deviceId: string;
  lastHeartbeat: Date;
}

class CrossDeviceTimerManager {
  async startSession(habitId: string): Promise<TimerState> {
    // Check for existing active sessions
    const activeSession = await this.checkActiveSession(habitId);
    
    if (activeSession && this.isSessionActive(activeSession)) {
      // Prompt user about existing session
      const userChoice = await this.promptConcurrentSession(activeSession);
      
      switch (userChoice) {
        case 'continue_existing':
          return this.resumeSession(activeSession);
        case 'start_new':
          await this.endSession(activeSession.sessionId);
          break;
        case 'cancel':
          throw new UserCancelledError();
      }
    }
    
    // Start new session
    const newSession: TimerState = {
      sessionId: this.generateUUID(),
      habitId,
      startedAt: new Date(),
      floorSeconds: await this.getFloorSeconds(habitId),
      currentPhase: 'floor',
      elapsedSeconds: 0,
      deviceId: await this.getDeviceId(),
      lastHeartbeat: new Date()
    };
    
    await this.persistTimerState(newSession);
    await this.startHeartbeat(newSession.sessionId);
    
    return newSession;
  }
  
  private isSessionActive(session: TimerState): boolean {
    const heartbeatAge = Date.now() - session.lastHeartbeat.getTime();
    return heartbeatAge < 5 * 60 * 1000; // 5 minutes
  }
  
  private async startHeartbeat(sessionId: string): Promise<void> {
    this.heartbeatIntervals.set(sessionId, setInterval(async () => {
      await this.updateHeartbeat(sessionId);
    }, 30 * 1000)); // 30 second heartbeat
  }
}
```

```typescript
interface ProofConfig {
  method: 'self_attest' | 'app_foreground' | 'photo_hash' | 'wearable_hr';
  enabled: boolean;
  retention_hours: number; // 24-72 hours max
  upload_consent: boolean;
}

class ProofManager {
  async generateProof(method: string, sessionId: string): Promise<ProofResult> {
    switch (method) {
      case 'self_attest':
        return this.selfAttestProof();
      
      case 'app_foreground':
        return this.appForegroundProof(sessionId);
      
      case 'photo_hash':
        return this.photoHashProof();
      
      case 'wearable_hr':
        return this.wearableHRProof(sessionId);
      
      default:
        throw new UnsupportedProofMethodError(method);
    }
  }
  
  private async photoHashProof(): Promise<ProofResult> {
    if (!await this.hasPermission('camera')) {
      throw new PermissionDeniedError('Camera permission required');
    }
    
    // Show clear consent dialog
    const consent = await this.showProofConsentDialog({
      title: 'Photo Proof',
      message: 'Take a quick photo as proof. The image will be processed locally and only a hash will be stored for 24 hours.',
      permissions: ['camera'],
      retention: '24 hours',
      privacy_note: 'The actual photo is never uploaded or stored permanently.'
    });
    
    if (!consent) {
      throw new UserDeclinedError();
    }
    
    try {
      const imageData = await this.capturePhoto();
      const hash = await this.hashImage(imageData);
      
      // Store hash locally with expiration
      await this.storeProofHash(hash, Date.now() + 24 * 60 * 60 * 1000);
      
      // Immediately delete image data
      imageData.dispose();
      
      return {
        method: 'photo_hash',
        result: true,
        metadata_hash: hash,
        timestamp: new Date()
      };
      
    } catch (error) {
      return {
        method: 'photo_hash',
        result: false,
        error: error.message,
        timestamp: new Date()
      };
    }
  }
  
  private async hashImage(imageData: ImageData): Promise<string> {
    // Use perceptual hashing to detect similar images while protecting privacy
    const features = await this.extractImageFeatures(imageData);
    return this.cryptoHash(features.join(''));
  }
  
  private async storeProofHash(hash: string, expiresAt: number): Promise<void> {
    const proofData = {
      hash,
      created_at: Date.now(),
      expires_at: expiresAt
    };
    
    await this.localDB.store('proof_hashes', hash, proofData);
    
    // Schedule cleanup
    setTimeout(() => {
      this.localDB.delete('proof_hashes', hash);
    }, expiresAt - Date.now());
  }
}
```

```typescript
class DesktopHelper {
  private activeWindowPoller: NodeJS.Timer | null = null;
  private hashedTitleCache: Map<string, string> = new Map();
  
  async startWindowMonitoring(sessionId: string): Promise<void> {
    if (!await this.hasPermission('accessibility')) {
      throw new PermissionError('Accessibility permission required');
    }
    
    this.activeWindowPoller = setInterval(async () => {
      const windowTitle = await this.getActiveWindowTitle();
      const hashedTitle = this.hashWindowTitle(windowTitle);
      
      await this.reportWindowActivity(sessionId, hashedTitle);
    }, 5000); // Check every 5 seconds
  }
  
  private async getActiveWindowTitle(): Promise<string> {
    // Platform-specific implementation
    if (process.platform === 'darwin') {
      return this.getMacActiveWindow();
    } else if (process.platform === 'win32') {
      return this.getWindowsActiveWindow();
    } else {
      return this.getLinuxActiveWindow();
    }
  }
  
  private hashWindowTitle(title: string): string {
    // Cache hashes to avoid recomputing
    if (this.hashedTitleCache.has(title)) {
      return this.hashedTitleCache.get(title)!;
    }
    
    const hash = crypto.createHash('sha256').update(title).digest('hex');
    this.hashedTitleCache.set(title, hash);
    
    return hash;
  }
  
  private async reportWindowActivity(sessionId: string, hashedTitle: string): Promise<void> {
    // Only report if user consented to upload
    const uploadConsent = await this.getUserUploadConsent();
    
    if (uploadConsent) {
      await this.api.reportProofActivity({
        session_id: sessionId,
        proof_type: 'desktop_window',
        activity_hash: hashedTitle,
        timestamp: new Date().toISOString()
      });
    } else {
      // Store locally only
      await this.localDB.store('window_activity', {
        session_id: sessionId,
        activity_hash: hashedTitle,
        timestamp: Date.now()
      });
    }
  }
```

```kotlin
// Foreground service for timer reliability
class TimerForegroundService : Service() {
    private val notificationId = 1001
    private var timerJob: Job? = null
    
    override fun onStartCommand(intent: Intent?, flags: Int, startId: Int): Int {
        val sessionData = intent?.getParcelableExtra<SessionData>("session")
        sessionData?.let { startTimer(it) }
        
        return START_STICKY // Restart if killed by system
    }
    
    private fun startTimer(session: SessionData) {
        createNotificationChannel()
        startForeground(notificationId, createTimerNotification(session))
        
        timerJob = CoroutineScope(Dispatchers.Main).launch {
            while (session.remainingSeconds > 0) {
                delay(1000)
                session.remainingSeconds--
                updateNotification(session)
                
                if (session.remainingSeconds == 0) {
                    onTimerComplete(session)
                    break
                }
            }
        }
    }
    
    private fun createTimerNotification(session: SessionData): Notification {
        return NotificationCompat.Builder(this, TIMER_CHANNEL_ID)
            .setContentTitle("${session.habitName} - Floor Timer")
            .setContentText("${session.remainingSeconds}s remaining")
            .setSmallIcon(R.drawable.ic_timer)
            .setOngoing(true)
            .setProgress(session.floorSeconds, session.remainingSeconds, false)
            .build()
    }
    
    // DND integration
    private fun enableDND() {
        val notificationManager = getSystemService(Context.NOTIFICATION_SERVICE) as NotificationManager
        
        if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.M) {

                )

                // Prompt user for permission

        }
    }
```
